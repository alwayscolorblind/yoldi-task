Тестовое задание для вакансии фронтенд разработчик в Yoldi
## Getting Started

Установка пакетов
```
yarn
```

Запуск
```
yarn dev
```

Для запросов:
- swr
- axios
- nextjs server actions

Необычно:
- Использовался react canary для использования хука ``useFormState``
- Генерация из OAS 3.0 при помощи ``rapini`` с конфигом ``swr``
